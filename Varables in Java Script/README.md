# ✅ JavaScript `var`, `let`, and `const` (Simple Notes)

JavaScript provides three ways to declare variables:

| Keyword | Scope | Redeclare | Reassign | Hoisting |
|---------|-------|-----------|----------|----------|
| `var`   | Function scope | ✅ Yes | ✅ Yes | ✅ Hoisted (initialized as `undefined`) |
| `let`   | Block scope | ❌ No | ✅ Yes | ✅ Hoisted (not initialized) |
| `const` | Block scope | ❌ No | ❌ No | ✅ Hoisted (not initialized) |
